# Ciclo de Vida de los Defectos

## ¿Qué va aquí?

Un documento Word o PDF que documenta el ciclo de vida completo de cada defecto encontrado durante el proyecto, cubriendo las siguientes etapas para cada uno:

1. Identificación — qué es el defecto, dónde se encontró y cómo se detectó
2. Clasificación — severidad, prioridad y tipo de prueba que lo descubrió
3. Seguimiento y validación — qué se hizo para resolverlo
4. Cierre — cómo se verificó que quedó resuelto y qué test lo confirma
5. Reflexión — qué aprendizaje dejó este defecto
